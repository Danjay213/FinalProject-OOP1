## 📖 About the Game

**Quiz Royale** is a turn-based educational battle game where players fight enemies by answering quiz questions correctly. Every correct answer deals damage to the enemy — but every wrong answer costs you your health.

Built with a **Java GUI**, the game features clickable answer buttons, health bars, and dynamic battle messages to bring the experience to life.

---

## 🎮 Core Gameplay Loop

```
1. ❓ A question appears on screen
2. 🖱️  The player selects an answer
3. ✅ If correct   → Enemy loses HP
   ❌ If wrong     → Player loses HP
4. 💀 If the enemy is defeated → Player gains Gold
5. ➡️  Next question appears
6. 🔁 Battle continues until one side reaches 0 HP
```

---

## 🏆 Win / Loss Conditions

| Condition | Requirement |
|-----------|-------------|
| 🏅 **Win** | Defeat all enemies by answering questions correctly |
| 💔 **Loss** | Player HP drops to 0 |
